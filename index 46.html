❓ FAQ

Got a question?

Does the image change visually? →
No — the image stays 100% clean. Title and keywords are only placed inside the file data as XMP, not on top of the image.
Does it work with JPG? →
The tool works with PNG, JPG, and WEBP. But PNG is best for metadata since JPG metadata gets stripped on most platforms.
How do I verify the metadata? →
Windows: Right-click → Properties → Details. Mac: Get Info. Photoshop: File → File Info. Or use exifdata.com.
Does it work for Etsy? →
Etsy strips metadata but the tool helps you with two features: SEO-friendly filename and auto-generating 13 Etsy tags + listing title.
Can I process multiple images at once? →
Yes! Upload unlimited images at once, set the title and keywords, and all are processed automatically with a progress bar and batch download. (Pro plan)
Is my data stored on a server? →
No — all processing happens directly in your browser. Your images are never uploaded to any server. Your privacy is 100% protected.
